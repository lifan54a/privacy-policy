隐私政策

Privacy Policy

最后更新日期：2026年3月4日

1. 引言

Jalan-Jalan China（以下简称"我们"）高度重视用户的隐私保护。本隐私政策说明我们如何收集、使用、存储和保护通过 Meta（Facebook/Instagram）平台获取的信息。

2. 我们收集的信息

我们通过 Meta API 收集以下信息：

  • 账号信息：页面/账号名称、ID、简介
  • 内容数据：发布的帖子、视频、图片
  • 洞察数据：播放量、互动量、曝光量
  • 受众数据：粉丝数量（不包含个人身份信息）

3. 信息使用目的

  • 生成社媒运营日报和分析报告
  • 分析内容表现和受众互动情况
  • 优化内容策略和发布计划

4. 信息存储与安全

  • 所有数据仅存储在本地服务器
  • 不与第三方共享或出售数据
  • 采用加密技术保护 API 凭证
